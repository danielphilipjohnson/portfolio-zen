Built Mistfall, a Drizzle-inspired schema DSL and browser database runtime designed to make IndexedDB predictable and developer-friendly. Introduces declarative tables, foreign keys, computed indexes, schema signatures, and fully typed CRUD operations removing the usual boilerplate and complexity of raw IndexedDB.

Implemented a unified IndexedDB + in-memory adapter with transactions, identity generation, referential integrity checks, typed predicates, computed fields, and schema-diff migrations creating a predictable, testable local data layer for complex React/Next.js applications.
